Verifiers for mobile wallets with SIOPV2-OIDC4VP

This is a bridge between the SIOPv2/OIDC4VP verifier flow and a standard OpenID authorization code flow and implicit flow. For web2 applications with all frameworks and popular CMS. This verifier supports EBSI V3 compliant wallets
  • Integration with standard OIDC lib (NodeJS, Python, PHP, C, C+, GO, Ruby,..Webflow, Wordpress,..;)
  • Resquest as a jwt and as an uri
  • Personal definition uri
  • PKCE
  • Client id as a DID or uri
  • JWT and JSON-LD
  • did;web, did:ethr, did:hedera, did:ebsi, did:tz, did:key
  • P-256, secp256k1, Ed25519, RS-256
  • Dedicated external resolver
  • PEX 2.0: rule count, all, array content, string pattern
